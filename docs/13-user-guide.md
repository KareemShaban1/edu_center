## كيف أبدأ؟

المنصة تربط **الإدارة** و**المعلمين** و**الطلاب** و**أولياء الأمور** في مكان واحد.

### تسجيل الدخول

| نوع الحساب | كيف تدخل |
|------------|----------|
| **الإدارة أو المساعد** | من رابط الدخول الذي يزودك به المركز التعليمي |
| **المعلم** | من نفس رابط دخول المركز |
| **الطالب** | من صفحة دخول الطلاب (رابط التسجيل من المركز) |
| **ولي الأمر** | من صفحة دخول أولياء الأمور (رابط التسجيل من المركز) |

أدخل **البريد الإلكتروني** و**كلمة المرور** التي حصلت عليها من إدارة المركز، ثم اضغط **تسجيل الدخول**.

![شاشة تسجيل دخول الطالب](/images/student_login.png)

### التنقل داخل المنصة

- على **الحاسوب**: استخدم **القائمة الجانبية** على يمين أو يسار الشاشة للانتقال بين الصفحات.
- على **الجوال**: اضغط أيقونة **القائمة** ☰ في الأعلى، أو استخدم **الأزرار السفلية** للوصول السريع.
- لتغيير اللغة: اضغط زر **العربية / English** في أعلى الصفحة.

## لوحة الإدارة

![نظرة على لوحة الإدارة](/images/admin_dashboard.png)

لوحة الإدارة هي المكان الذي تدير منه المركز التعليمي بالكامل: الطلاب، المعلمين، الحصص، الحضور، الدرجات، الرسوم، والإعلانات.

### الصفحة الرئيسية

تعرض لك ملخصاً سريعاً عن حالة المركز: عدد الطلاب والمعلمين، نسبة الحضور، الرسوم المتأخرة، وأحدث الإعلانات.

### إدارة الأشخاص

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **الطلاب** | إضافة طلاب جدد، تعديل بياناتهم، ربطهم بالصف والفصل والمجموعة |
| **المعلمين** | إضافة معلمين وتحديث بياناتهم وربطهم بالمجموعات الدراسية |
| **أولياء الأمور** | إدارة حسابات أولياء الأمور وربط كل ولي أمر بأبنائه |

### الهيكل الدراسي

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **الصفوف** | إنشاء الصفوف الدراسية (مثل: الصف الأول الثانوي) |
| **الفصول** | إنشاء الفصول داخل كل صف |
| **المجموعات** | إنشاء مجموعات الطلاب وعرض حصص كل مجموعة |

**ترتيب الإعداد المقترح:** الصفوف ← الفصول ← المجموعات ← ثم إضافة الطلاب والمعلمين.

### المنهج والواجبات

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **الوحدات** | تنظيم وحدات المنهج الدراسي |
| **الدروس** | إضافة الدروس داخل كل وحدة |
| **الواجبات** | إنشاء واجبات للطلاب ومراجعة ما سلّموه وإضافة ملاحظات للمعلم |

### الحصص والحضور والدرجات

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **الحصص** | جدولة الحصص الدراسية (حضورية أو عبر الإنترنت) |
| **الحضور** | اختر المجموعة ← سجّل حضور وغياب الطلاب لليوم أو لتاريخ سابق ← راجع السجل |
| **الامتحانات** | سجّل درجات الامتحانات لكل مجموعة وتاريخ |
| **الكويزات** | سجّل درجات الاختبارات القصيرة (الكويز) |

**خطوات تسجيل الحضور أو الدرجات:**
1. افتح **الحضور** أو **الامتحانات** أو **الكويزات** من القائمة.
2. اختر **المجموعة** المناسبة.
3. اختر **التاريخ** (اليوم أو تاريخ سابق).
4. املأ الجدول لكل طالب.
5. اضغط **حفظ**.

### الرسوم والمدفوعات

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **الرسوم** | تحديد أنواع الرسوم ومبالغها للطلاب |
| **المدفوعات** | تسجيل ما دفعه كل طالب (اليوم أو تاريخ سابق) ومتابعة السجل |

### التواصل والمحتوى

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **المكتبة** | رفع ملفات ومواد تعليمية يستفيد منها المعلمون والطلاب |
| **الإعلانات** | نشر إعلانات عامة لجميع مستخدمي المركز |
| **الإشعارات** | إرسال تنبيهات للطلاب وأولياء الأمور على أجهزتهم |
| **واتساب** | إرسال رسائل واتساب للطلاب أو أولياء الأمور |
| **قوالب واتساب** | إنشاء رسائل جاهزة تُرسل بسرعة (مثل: تذكير بالحضور أو الرسوم) |

### الشهادات والتقارير

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **الشهادات** | إصدار شهادات للطلاب من قوالب جاهزة |
| **قوالب الشهادات** | تصميم شكل الشهادة قبل إصدارها |
| **التقارير** | عرض تقارير الحضور والامتحانات والكويزات والمدفوعات |

### إعدادات المركز

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **صفحات التسويق** | إنشاء صفحات عامة للتعريف بالمركز أو بالمعلمين أو الدورات |
| **المستخدمين** | إدارة حسابات الدخول للموظفين |
| **الأدوار والصلاحيات** | تحديد ما يستطيع كل موظف الوصول إليه |
| **الإعدادات** | اسم المركز، بيانات التواصل، اللغة، والفصل الدراسي |

## لوحة المعلم

![نظرة على لوحة المعلم](/images/teacher_dashboard.png)

المعلم يتابع مجموعاته الدراسية ويسجّل الحضور والدرجات ويرفع الواجبات.

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **الرئيسية** | نظرة سريعة على حصصك وطلابك |
| **حصصي** | عرض المجموعات المعيّنة لك وقائمة الطلاب في كل مجموعة |
| **الحصص** | عرض جدول الحصص والدخول للحصة المباشرة عبر الفيديو |
| **الحضور** | تسجيل حضور وغياب طلابك |
| **الامتحانات** | إدخال درجات الامتحانات |
| **الكويزات** | إنشاء وتصحيح الاختبارات القصيرة |
| **الواجبات** | تعيين واجبات ومراجعة ما رفعه الطلاب |
| **المكتبة** | تحميل المواد التعليمية المشتركة |

**نصيحة للمعلم:** ابدأ يومك من **الحصص** أو **الحضور** للمجموعة التي تدرّسها اليوم.

## لوحة الطالب

![نظرة على لوحة الطالب](/images/student_dashboard.png)

الطالب يتابع تقدمه الدراسي ويحضّر الحصص ويسلّم الواجبات.

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **الرئيسية** | ملخص: حصصك، نسبة حضورك، معدلك، والواجبات المعلقة |
| **حصصي** | عرض جدول الحصص والانضمام للحصة المباشرة عند بدئها |
| **الحضور** | مراجعة سجل حضورك وغيابك |
| **الدرجات** | عرض نتائج الامتحانات والكويزات |
| **الواجبات** | عرض الواجبات المطلوبة ورفع ملفاتك وإضافة ملاحظات |
| **المكتبة** | تحميل المواد الدراسية |
| **الشهادات** | عرض وتحميل الشهادات التي حصلت عليها |

**للتسجيل لأول مرة:** إذا كان مركزك يسمح بالتسجيل الذاتي، استخدم رابط **تسجيل طالب جديد** من صفحة الدخول.

## لوحة ولي الأمر

![نظرة على لوحة ولي الأمر](/images/parent_dashboard.png)

ولي الأمر يتابع أبناءه دراسياً ومالياً من مكان واحد.

| الصفحة | ماذا تفعل؟ |
|--------|-----------|
| **الرئيسية** | ملخص سريع لكل ابن: الحضور، الرسوم، وآخر المستجدات |
| **الأبناء** | تفاصيل كل ابن: بياناته، حضوره، ونتائجه |
| **الحضور** | سجل حضور وغياب جميع أبنائك |
| **الامتحانات** | نتائج امتحانات الأبناء |
| **الكويزات** | نتائج الاختبارات القصيرة |
| **الرسوم** | الرسوم المستحقة والمدفوعة لكل ابن |
| **التقارير** | تقرير شامل يجمع الحضور والرسوم والدرجات |

**ملاحظة:** إذا كان لديك أبناء في أكثر من مركز تعليمي، اختر المركز المناسب بعد تسجيل الدخول.

---

## نصائح مفيدة

- **احفظ بياناتك:** لا تشارك بريدك الإلكتروني أو كلمة المرور مع أي شخص.
- **على الجوال:** يمكنك تثبيت المنصة كتطبيق من زر **تثبيت التطبيق** في القائمة الجانبية للوصول السريع.
- **الإشعارات:** راقب أيقونة **الجرس** في أعلى الصفحة للتنبيهات الجديدة.
- **تحتاج مساعدة؟** تواصل مع إدارة مركزك، أو ارجع لهذا الدليل من رابط **دليل المستخدم** في الصفحة الرئيسية للمنصة.
